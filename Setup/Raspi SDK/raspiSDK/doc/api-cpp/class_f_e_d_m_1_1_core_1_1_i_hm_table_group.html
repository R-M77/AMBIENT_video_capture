Detailed Description

Internal class providing Host Mode table specific methods.

Description
This class realizes the nested interface with special methods for Host Mode table for the Reader class ReaderModule.
Methods of this class can be called with the IHmTable member of ReaderModule.

◆ SetLength()

int IHmTableGroup::SetLength ( unsigned int  uiLength)

Set number of valid data items in the table, if the application has built the table in front of transmit actions.

Parameters
[in]uiLengthNumber of valid table items in the Host Mode Table
Returns
OK (=0) or error code (<0)

Description
This method is normally not used.
In rare cases, when the Host Mode Table is built for non-addressed data transfer to the Transponder, the table can be set to 1.
Non-addressed communication with Transponders need no Inventory and the table keeps empty.

◆ Reset()

int IHmTableGroup::Reset ( unsigned int  uiDataFlags = 0)

Reset length of valid data items in the table.

Parameters
[in]uiDataFlagsFlags to indicate the table members to be initialized to or 0 (preset) to initialize no members
Returns
OK (=0) or error code (<0)

Description
After call of this method, GetLength() will return 0. In almost all cases, the transferred parameter uiDataFlags can be set to 0 (preset).

Note
A call of an Inventory command does not clear the internal Host Mode Table. Thus, the call of Reset() have to be done in front of each Inventory.

◆ GetItemByIndex()

IHmTableGroup::HmTableItem * IHmTableGroup::GetItemByIndex ( unsigned int  uiIdx)

Get a valid table item, identified by index.

Description

Parameters
[in]uiIdxTable index, beginning with 0
Returns
table item. See HmTableItem for more information

Description
With this method, the table items (each contains one record from the Inventory response) can be gradually requested.
The following example demonstrate the direct table access method.
For more efficient and modern coding, the concept of TagHandlers should be preferred. See Inventory for a simple example.


Example:

using namespace FEDM::Core;
IHmTableGroup::HmTableItem* pItem = NULL; // pointer to table item
reader.IHmTable.Reset();
reader.ICmd.SetCommandPara(ReaderCommand::_0xB0::SUB_COMMAND, (unsigned char)0x01);
reader.ICmd.SetCommandPara(ReaderCommand::_0xB0::SubCmd::_0x01::Req::MODE, (unsigned char)0x00);
int back = reader.ICmd.SendProtocol(0xB0);
if( back == 0 || // status: OK
back == 0x84 || // status: RF warning
back == 0x94 ) // status: more data
{
// internal Host Mode Table contains now the new records
for(int iIdx=0; iIdx<reader.IHmTable.GetLength(); iIdx++)
{
pItem = reader.IHmTable.GetItemByIndex(iIdx);
if(pItem->m_bIsIDD)
{
// IDD is part of the received record
}
}
}
NOTE: reader is an object of type FEDM::Core::ReaderModule
