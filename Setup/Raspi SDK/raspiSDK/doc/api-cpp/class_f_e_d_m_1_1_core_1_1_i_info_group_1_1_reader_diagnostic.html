Detailed Description

Class for collecting all reader diagnostic information.

Description
This class contains all data values, which are potentially part of the Reader's response of the command [0x6E] Reader Diagnostic.

std::string IInfoGroup::ReaderDiagnostic::GetPartialReport ( unsigned char  ucMode) const

Returns a partial report with Reader diagnostic information.

Parameters
[in]ucModemode byte in command [0x6E] Reader Diagnostic
Returns
string with partial report

Description
This method returns a partial report, based on information which must be previously read from the Reader with the command [0x6E] Reader Diagnostic.
Use SendProtocol or ReadReaderDiagnostic to read the information.
See Reader's System Manual for the list with all supported mode bytes.
