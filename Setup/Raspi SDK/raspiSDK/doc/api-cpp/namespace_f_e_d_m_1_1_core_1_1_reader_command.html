Detailed Description

Namespace ReaderCommand collects all command parameters.

Description
This namespace collects command parameters to be used with ICommandGroup::SetCommandPara or ICommandGroup::GetCommandPara.
These methods can be called with the ICmd member of the Reader class ReaderModule.

Note
It is highly recommended to read the Reader's System Manual when coding the Reader communication with these command parameters.
