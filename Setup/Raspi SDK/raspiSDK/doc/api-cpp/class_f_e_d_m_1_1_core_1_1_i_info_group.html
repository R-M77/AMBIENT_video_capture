Detailed Description

Internal class providing Reader info specific methods.

Description
This class realizes the nested interface with special methods for requesting general information for the Reader class ReaderModule.
Methods of this class can be called with the IInfo member of ReaderModule.

◆ ReadReaderInfo()

int IInfoGroup::ReadReaderInfo ( unsigned int  uiProtocolFrame = FEDM_PRT_FRAME_STANDARD)

Read complete reader info and set the Reader Type.

Parameters
[in]uiProtocolFrametype of protocol frame (preset to STANDARD) to be used for the communication.
Returns
  • 0 if OK
  • < 0 error code
  • > 0 status byte of the responded protocol, if the Reader signals a problem. See Reader's System Manual for the full list with all status bytes.

Description
ReadReaderInfo() is one of the important methods, that have to be called first after connecting to the Reader.
This method sets the Reader Type internally and can set the protocol frame properly to the connected Reader.
This method is normally called in the connect methods internally (see IPortGroup), but in some cases, e.g. when one of the Connect methods is called without the detect option, the ReadReaderInfo() must be called separately.
