◆ OnNewNotification()

virtual void FEDM::Core::INotificationListener::OnNewNotification ( int  iError,
std::string  sRemoteIP,
unsigned int  uiPortNumber 
)
pure virtual

Is called, when Reader is working in Notification-Mode and sends a notification with Transponder data.

Parameters
[in]iErrorerror code (< 0) or OK (=0) or Status Byte (>0) from notification protocol
[in]sRemoteIPIP-Address of the Reader, which sent the notification
[in]uiPortNumberTCP-Port, which receives the notification

If iError=0, the response data are collected in the internal Buffered Read Mode Table and must be requested gradually with IBrmTableGroup::GetItemByIndex()
