Detailed Description

Internal class providing methods for firmware update.

Description
This class realizes the nested interface with methods for firmware update for the Reader class ReaderModule.
Methods of this class can be called with the IFwUpd member of ReaderModule.

Note
It is strongly recommended to read and follow the instructions in the Application Note N60810-e-ID-B.pdf

Method to execute a firmware update.

Parameters
[in]sXmlFwPathnameFirmware pathname (path and filename). Firmware files are XML files and available from FEIG ELECTRONIC
[in]uiLogOptionLogging option
[in]sLogDirOptional destination path for log files
[in]pListener(optional) Pointer to listener class in application, which must be derived from IFwUpdListener
Returns
error code (<0) or OK (=0)

Description
The update is executed in silent mode with optional update messages reflected with a listener.
Different logging options can be enabled:
a) no logging: set uiLogOption = IFwUpdGroup::LOGGING_OFF
b) standard logging: set uiLogOption = IFwUpdGroup::LOGGING_STANDARD (this option is recommended)
c) standard logging and additional protocol logging: set uiLogOption = IFwUpdGroup::LOGGING_WITH_PROTOCOLS

Note
The connection to the Reader must be established before the use of this update method.
