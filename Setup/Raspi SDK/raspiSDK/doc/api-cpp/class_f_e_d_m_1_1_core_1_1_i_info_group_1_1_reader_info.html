Detailed Description

Class for collecting all reader information.

Description
This class contains all data values, which are potentially part of the Reader's response of the command [0x66] Reader Info.

◆ GetPartialReport()

std::string IInfoGroup::ReaderInfo::GetPartialReport ( unsigned char  ucMode) const

Returns a partial report with Reader information.

Parameters
[in]ucModemode byte in command [0x66] Reader Info
Returns
string with partial report

Description
This method returns a partial report, based on information which must be previously read from the Reader with the command [0x66] Reader Info.
Use SendProtocol or ReadReaderInfo to read the information.
See Reader's System Manual for the list with all supported mode bytes.
