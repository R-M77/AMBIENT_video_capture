Detailed Description

Transponder class, nested in class TH_ISO14443_4_MIFARE_DESFire, providing manufacturer specific commands for ISO 14443-4 compliant Transponders of type MIFARE DESFire from manufacturer NXP Semiconductors.

Description
High-level class for easy-to-use Transponder specific communication. See FEDM::Core::ITagGroup for additional information.
This class implements the communication with [0xC1] Reader commands and software cryptographic.
Detailed description about the command interface can be found in:

  • Product Datasheet for MIFARE DESFire from NXP Semiconductors
  • System Manual SoftCrypto Functions from FEIG ELECTRONIC
Note
Programmers should be familiar with the ISO 14443 and ISO/IEC 7816 standards
