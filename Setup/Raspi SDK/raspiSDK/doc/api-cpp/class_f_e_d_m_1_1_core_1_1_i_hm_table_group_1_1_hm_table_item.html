Detailed Description

Class represents one valid record of requested Host Mode data, initially created with the command [0xB0][0x01] Inventory.

Description
This class contains all record members, which are potentially part of the Reader's response of the command [0xB0][0x01] Inventory.
An item can be requested after the command was executed successfully with the method FEDM::Core::IHmTableGroup::GetItemByIndex().

The example demonstrate the direct table access method. For more efficient and modern coding, the concept of TagHandlers should be preferred. See Inventory for a simple example.

The class can also buffer request or responded data for the following basic Transponder commands:
[0xB0][0x23] Read Multiple Blocks
[0xB0][0x24] Write Multiple Blocks
[0xB0][0x25] Select
[0xB0][0x27] Write AFI (for ISO 15693)
[0xB0][0x29] Write DSFID (for ISO 15693)
[0xB0][0x2B] Get System Information (for ISO 15693)
[0xB0][0x2C] Get Multiple Block Security Status (for ISO 15693)
[0xB2][0x2B] ISO 14443-4 Transponder Info

◆ GetRSSI()

const IHmTableGroup::RSSI_MAP & IHmTableGroup::HmTableItem::GetRSSI ( )

Method returns map with Received Signal Strength Indication (RSSI)

Returns
map with RSSI value and status for each antenna

Description
Method to query a map with Received Signal Strength Indication (RSSI). The map contains a RSSI and status value for each antenna, where the Transponder is detected. The key of the map is the antenna number, the value is a pointer to a structure of type RssiItem with the RSSI and status value.
