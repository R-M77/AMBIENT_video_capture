)

Method to add a new Function-Unit as a child.

Parameters
[in]uiOutputNoNumber of the output, where the Function-Unit is connected
[in]uiFunctionUnitTypeType of Function-Unit (TYPE_HF_MULTIPLEXER, TYPE_HF_DYNAMIC_ANTENNA_TUNER, TYPE_UHF_MULTIPLEXER)
Returns
  • 0 if OK
  • < 0 error code

Description
This method creates a new Function-Unit of type uiFunctionUnitType and adds this object to the internal child list.
The cascade level is automatically set.
Use GetChild to request the new created Function-Unit object.
